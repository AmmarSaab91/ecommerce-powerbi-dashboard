# E-Commerce Power BI Dashboard

## Overview

This project analyzes e-commerce data using Power BI, focusing on
revenue, customer behavior, and product performance.

## Datasets size(rows,columns):

* Customers: (8000,20)
* Orders: (250000,28)
* Product Summary: (140,9)
* Monthly Revenue: (75,10)

## Key Metrics

* Total Revenue (Delivered): $2.59M
* Total Customers: 8,000
* Total Orders: 25,000
* Cancellation Rate: 5.82%
* Average Order Value: $125.68

## Analysis Sections

* Revenue \& Sales Analysis
* Customer Analysis
* Geographic Analysis

## Key Insights

* Top products drive a large share of revenue.
* Electronics and Clothing dominate sales.
* The United States is the top market.
* Low cancellation rate indicates strong order completion.
* Customer behavior varies by device and payment method.

## Tools Used

* Power BI
* DAX
* Data Modeling
